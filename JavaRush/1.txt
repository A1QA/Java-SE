Каждая переменная в Java имеет три важных свойства: тип, имя и значение.

- Имя используется для того, чтобы отличать одну переменную от другой. Это как надпись на коробке.

- Тип переменной определяет тип значений/данных, которые в ней можно хранить. 
  В коробке для торта храним торт, в коробке для обуви – туфли, и т.д.

- Значение – это некий объект, данные или информация, которая хранится в переменной.

int a;	Создаем переменную по имени a типа int.
2	String s;	Создаем переменную по имени s типа String.
3	double c;	Создаем переменную по имени c типа double.
